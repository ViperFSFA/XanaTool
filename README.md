# **XanaTool - Community Edition**  
### 🔥 Created by ViperFSFA  

XanaTool is a **real-time network packet sniffer and analyzer** designed for cybersecurity professionals, ethical hackers, and researchers. With its **intuitive GUI** built using **Tkinter**, this tool enables users to **monitor, decode, and inspect network traffic** with ease.  

> **⚠️ Disclaimer:**  
> - Use XanaTool **ONLY** on networks you own or have explicit permission to analyze.  
> - Unethical use, including unauthorized packet sniffing, is **strictly prohibited**.  
> - This software is provided **as-is** with **no warranties**.  

---

## 🚀 Features  

✅ **Live Packet Sniffing** – Capture packets in real-time.  
✅ **Protocol Filtering** – Filter traffic by TCP, UDP, ICMP, DNS, and more.  
✅ **Deep Packet Inspection** – Decode packets and analyze headers/payloads.  
✅ **Sensitive Data Detection** – Highlight credentials, tokens, and key data.  
✅ **Custom Packet Injection** – Manually send crafted packets for testing.  
✅ **Activity Logging** – Keeps track of all interactions and captured data.  
✅ **User-Friendly UI** – Modern dark-themed interface for easy navigation.

## 🎯 How to Use XanaTool  

### **🔴 Step 1: Start Sniffing Packets**  
- Click **"Start Capture"** to begin monitoring network traffic.  
- The tool will display incoming/outgoing packets in real-time.  

### **🔵 Step 2: Apply Filters (Optional)**  
- Use the **Protocol Filter** (TCP, UDP, DNS, ICMP, etc.) to focus on specific traffic types.  
- Enter **custom keywords** to filter packets by content.  

### **🟢 Step 3: Inspect Packets**  
- Click on any captured packet to view its **detailed breakdown**.  
- Examine **headers, payloads, and potential sensitive data**.  

### **🟣 Step 4: Inject Custom Packets (Advanced Users)**  
- Go to the **"Packet Injection"** tab.  
- Enter **custom TCP/UDP payloads** and destination details.  
- Click **"Send Packet"** to transmit your crafted packet.
